In order to run this code, navigate into the "code" directory and run the following commands on the terminal:

-----------
cmake .
make
-----------

Once the project has been compiled, run the code as specified in the Assignment Document. 
However, the executable is called 'Project3’.

Example Command line run:

./Project3 BASE_NAME INPUT_DIRECTORY

All output will be posted to the command line. This information will be labelled.
Once the data for the feature curve is outputted, copy and paste that information into the input box of ‘http://www.shodor.org/interactivate/activities/SimplePlot/'.
